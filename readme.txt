== Description ==

= Buddypress statistic bar! =

This Wordpress plugin installs a **new sidebar widget** that can display the **buddypress community statistic**.


== Installation ==

1. Unzip bp-statistic-bar.zip
2. Upload the `bp-statistic-bar` directory and all its contents into the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Use the widget admin page to add the widget to one of your sidebars and configure it
